# Z.AI Project Configuration - Crypto Analytics Dashboard Pro

## 🎯 Project Overview
- **Name**: Crypto Analytics Dashboard Pro
- **Type**: Cryptocurrency Analytics & Trading Platform
- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript 5
- **Database**: SQLite + Prisma ORM

## 🏗 Architecture Overview

### Frontend Structure
```
src/
├── app/page.tsx                    # Main dashboard page
├── app/api/                       # API routes
│   ├── ai-analysis/route.ts       # AI analysis endpoint
│   ├── dashboard/route.ts         # Dashboard data endpoint
│   ├── trading-signals-fast/route.ts
│   └── alerts-fast/route.ts
├── components/
│   ├── ui/                        # shadcn/ui components
│   ├── dashboard/                 # Dashboard-specific components
│   │   ├── EnhancedMultiLayerDashboard.tsx
│   │   └── EnhancedAIAnalysisPanel.tsx
│   └── CoinManagementPanel.tsx
├── lib/                          # Core services
│   ├── enhanced-ai-analysis-service.ts  # Main AI service v2.0
│   ├── crypto-service.ts                # Market data service
│   ├── trading-signals.ts               # Signal generation
│   ├── db.ts                           # Database client
│   ├── socket.ts                       # WebSocket handling
│   └── ai-logger.ts                    # AI logging
└── hooks/                        # Custom React hooks
```

### Database Schema (Prisma)
- **Cryptocurrency**: Crypto metadata and settings
- **PriceHistory**: Historical price data
- **OnChainMetrics**: MVRV, NUPL, SOPR, etc.
- **TechnicalIndicators**: RSI, MA, MACD, etc.
- **SentimentMetrics**: Fear & Greed, social sentiment
- **DerivativeMetrics**: Open interest, funding rates
- **AnalysisHistory**: AI analysis results storage

## 🔑 Key Components

### 1. EnhancedAIAnalysisService (Main AI Engine)
**Location**: `src/lib/enhanced-ai-analysis-service.ts`
**Purpose**: Multi-timeframe AI analysis with 47+ indicators
**Key Methods**:
- `performEnhancedAnalysis(coinId, analysisType)`
- `generateEnhancedFallbackAnalysis(coinId)`
- `executeAnalysisWithClearErrorHandling(context, analysisType)`

### 2. CryptoDataService (Market Data)
**Location**: `src/lib/crypto-service.ts`
**Purpose**: Aggregate market data from various sources
**Key Methods**:
- `getCompleteCryptoData(coinId)`
- `getHistoricalData(coinId, timeframe)`
- `getOnChainMetrics(coinId)`

### 3. TradingSignalService (Signal Generation)
**Location**: `src/lib/trading-signals.ts`
**Purpose**: Generate trading signals with risk assessment
**Key Methods**:
- `generateSignal(marketData)`
- `calculateRiskLevel(signal)`
- `validateSignal(signal)`

## 🤖 AI Integration

### AI Providers
1. **Z.AI** (Primary): `z-ai-web-dev-sdk`
2. **OpenAI/ChatGPT** (Secondary): `openai` package
3. **Rule-based** (Fallback): Custom logic

### AI Analysis Flow
1. **Data Collection**: Gather 47+ market indicators
2. **Context Building**: Structure data for AI processing
3. **AI Processing**: Send to Z.AI/OpenAI with enhanced prompts
4. **Result Processing**: Parse and structure AI responses
5. **Risk Management**: Add risk assessment and position sizing
6. **Storage**: Save to database with metadata

### Prompt Templates
**Location**: `docs/AI_Prompt_templates.md`
**Features**:
- Version 2.0 enhanced prompts
- Multi-timeframe analysis structure
- Professional trading recommendations
- Risk management guidelines

## 🔌 API Endpoints

### Core APIs
```typescript
// AI Analysis (Enhanced)
POST /api/ai-analysis?action=analyze&analysisType=enhanced&coinId=bitcoin

// Dashboard Data
GET /api/dashboard?coinId=bitcoin

// Trading Signals
GET /api/trading-signals-fast?action=signal&coinId=bitcoin

// Market Alerts
GET /api/alerts-fast?action=process-data&coinId=bitcoin

// Cryptocurrency Management
GET /api/cryptocurrencies?activeOnly=true
```
